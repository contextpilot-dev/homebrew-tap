# ContextPilot Homebrew Tap

Official Homebrew tap for [ContextPilot](https://github.com/contextpilot-dev/contextpilot).

## Installation

```bash
brew tap contextpilot-dev/tap
brew install contextpilot
```

## Update

```bash
brew upgrade contextpilot
```

## Uninstall

```bash
brew uninstall contextpilot
brew untap contextpilot-dev/tap
```

## About ContextPilot

Make every AI tool understand your codebase. Generate and sync `.cursorrules`, `CLAUDE.md`, and `copilot-instructions.md`.
